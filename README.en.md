# 🀄 Chinese Official Mahjong Scoring Calculator

A clean and user-friendly scoring calculator for Chinese Official Mahjong, covering all 81 official scoring patterns.

[中文说明](./README.md)

## 🎮 Project Link

- Live site: [https://kkkkkkkkatie.github.io/mahjong-calculator/](https://kkkkkkkkatie.github.io/mahjong-calculator/)
- GitHub repository: [KKKKKKKKatie/mahjong-calculator](https://github.com/KKKKKKKKatie/mahjong-calculator)

## ✨ Features

- Responsive design for desktop and mobile
- Search all scoring patterns
- Filter by point value
- Real-time total score calculation
- Built-in explanations for each pattern

## 🚀 Deploy to GitHub Pages

### Option 1: Upload Directly

1. Create a new GitHub repository named `mahjong-calculator`
2. Upload all project files to the repository
3. Open repository Settings → Pages
4. Choose the `main` branch and `/ (root)` as the source
5. Save and wait for deployment to finish

### Option 2: Use Git Commands

```bash
# In the mahjong-calculator directory
git init
git add .
git commit -m "Initial commit: Chinese Official Mahjong Scoring Calculator"
git branch -M main
git remote add origin https://github.com/KKKKKKKKatie/mahjong-calculator.git
git push -u origin main
```

Then enable GitHub Pages in the repository settings.
